Для корректного использования модуля, он должен быть расположен в папке modules
Подгрузка модуля осуществляется javascript кодом:


//  подключение модуля загрузки файлов
    $("#load_img").load("modules/load_images/index.php",{label:'Загрузить',path:'../../../img/1/',func:'func1'});
    func1 = function(f){
        $("#load_img").append("<img src='../img/1/"+f+"'/>");
    };

    где
    load_img -  блок в который будет подгружаться форма выбора файла
    modules/load_images/index.php - адрес до скрипта
    label - загаловок формы загрузки
    path - путь к директории для загруженых файлов. (../../  -  вы окажетесь в той же дир. что и папка modules)
    func - имя функции которую надо вызвать по окончанию загрузки. (в примере я вывожу загруженное изображение в том же блоке что и форма загрузи)
